k-means algorithm

  • What are \(m\), \(\widehat{\overline{x}_k}\) and \(\widehat{\overline{x}_j}\)
  1. Find cluster means with cluster memberships fixed \[\begin{align*} \widehat{\overline{x}_j} = \text{argmin}_m \sum_{cluster(i) = j} ||x_i - m||^2 \end{align*}\]
  2. Find cluster memberships with cluster means fixed \[\begin{align*} \widehat{cluster(i)} = \text{argmin}_k ||x_i - \widehat{\overline{x}_k}||^2 \end{align*}\]

PCA parameters

  • Are there any parameters to tweak here?

  • PCA: No but there are choices in scaling or other PCA variants.

    • If uncentered: Lose the maximizing variance interpretation
    • If not scaling: Features with large scales contribute more to the variance.

  • Sparse PCA: Can use when \(p\) much bigger than \(n\)

    • Add sparsity penalty

  • Functional PCA: Vary over time and space

  • Robust PCA: Use the different metric

    • Squares metric (Variance based of sum of squares) can be sensitive to single points
    • Use a different metric, such as absolute value

MDS parameters

  • Are there any parameters to tweak here?

  • Can choose different stress function

    • Takes distance matrix gives output projections

  • Least squares or Kruskal-Shephard Scaling: \[\begin{align} \sqrt{\sum_{i \neq j}(d_{ij} - ||z_i - z_{j}||)^2} \end{align}\]

    • Used in cmdscale

  • Sammon mapping: normalizes for smaller distances \[\begin{align} \sqrt{\sum_{i \neq j}(\frac{d_{ij} - ||z_i - z_{j}||}{d_{ij}})^2} \end{align}\]

    • See MASS::sammon

  • Shepard-Kruskal nonmetric scaling: \[\begin{align} \sqrt{\sum_{i \neq j}(d_{ij} - g(||z_i - z_{j}||))^2} \end{align}\]

    • Where some \(g\) is an increasing function.
    • See isoMDS (think it uses a power transform here \(g(x) = x^q\), \(q\ge 1\))
